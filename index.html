基础环境配置

JDK 配置

Spark 基于 Scala 开发，JDK 是运行基础。推荐 JDK 8 稳定版，兼容性好且性能优异。

# 设置 JAVA_HOME 环境变量
export JAVA_HOME=/usr/local/jdk8
export PATH=$JAVA_HOME/bin:$PATH

# 验证安装
java -version

Hadoop 配置

提供 HDFS 存储与 YARN 资源调度，Spark 核心依赖组件。需配置 core-site.xml 和 hdfs-site.xml。

# 格式化 NameNode
hdfs namenode -format

# 启动 HDFS 服务
start-dfs.sh
# 启动 YARN 服务
start-yarn.sh

Zookeeper 配置

分布式协调服务，用于 Spark HA 高可用主备切换。配置 zoo.cfg 并启动集群。

# 启动 Zookeeper 服务
zkServer.sh start

# 查看状态
zkServer.sh status
# 连接客户端
zkCli.sh -server localhost:2181

部署模式

Local 模式

本地单线程/多线程运行，适合开发调试。无需集群环境，直接运行即可。

# 本地单线程
spark-submit --master local

# 本地 4 线程
spark-submit --master local[4]
开发调试 单节点

Standalone 模式

Spark 自带集群管理器，独立部署模式。包含 Master 和 Worker 节点配置。

# 启动 Master
start-master.sh

# 启动 Worker
start-worker.sh spark://master:7077

# 提交应用
spark-submit --master spark://master:7077
独立集群 Master-Slave

HA 高可用模式

基于 Zookeeper 实现 Master 主备切换，避免单点故障。生产环境推荐配置。

# 配置 spark-env.sh
export SPARK_DAEMON_JAVA_OPTS="-Dspark.deploy.recoveryMode=ZOOKEEPER"
export SPARK_DAEMON_JAVA_OPTS="-Dspark.deploy.zookeeper.url=zk1:2181,zk2:2181"
生产环境 故障转移

YARN 模式

运行在 Hadoop YARN 之上，利用现有集群资源。支持 Client 和 Cluster 两种提交方式。

# Client 模式（Driver 在本地）
spark-submit --master yarn --deploy-mode client

# Cluster 模式（Driver 在集群）
spark-submit --master yarn --deploy-mode cluster
资源调度 企业级

高阶配置

内存优化配置

合理配置 Executor 内存、堆外内存和序列化方式，避免 OOM 并提升性能。

# spark-defaults.conf
spark.executor.memory 4g
spark.executor.memoryOverhead 512m
spark.memory.fraction 0.8
spark.memory.storageFraction 0.3
spark.serializer org.apache.spark.serializer.KryoSerializer

并行度调优

根据集群资源调整并行度，优化数据分区，减少 Shuffle 开销。

# 默认并行度
spark.default.parallelism 200

# SQL 分区数
spark.sql.shuffle.partitions 200

# 自动广播 Join
spark.sql.autoBroadcastJoinThreshold 10MB

动态资源分配

根据工作负载动态增减 Executor 数量，提高集群资源利用率。

# 开启动态分配
spark.dynamicAllocation.enabled true
spark.dynamicAllocation.minExecutors 2
spark.dynamicAllocation.maxExecutors 20
spark.dynamicAllocation.initialExecutors 4
spark.shuffle.service.enabled true

实战案例

WordCount 词频统计

Spark 入门经典案例，使用 RDD 或 DataFrame API 实现文本词频统计。

val lines = sc.textFile("hdfs://input.txt")
val words = lines.flatMap(_.split(" "))
val pairs = words.map((_, 1))
val counts = pairs.reduceByKey(_ + _)
counts.saveAsTextFile("hdfs://output")

Spark SQL 数据分析

使用 Spark SQL 进行结构化数据处理，支持 Hive 集成和复杂查询。

val df = spark.read.json("people.json")
df.createOrReplaceTempView("people")

val result = spark.sql(""" |SELECT age, COUNT(*) as cnt |FROM people |GROUP BY age |ORDER BY cnt DESC |""".stripMargin)

Spark Streaming 实时处理

基于 Structured Streaming 实现实时数据流处理，支持 Kafka 数据源。

val df = spark
  .readStream
  .format("kafka")
  .option("subscribe", "topic")
  .load()

df.writeStream
  .outputMode("append")
  .format("console")
  .start()
